Apps

Movie Buffs

The first Group project from my Coding Bootcamp course. My team created an app which used two API's from Wikipedia and OMDb to display information regarding entertainment media.

Weather Dash

Weather Dash allows users to search up current weather conditions and a five-day forecast for any city they search for. This app uses two Open Weather's API and Moment.JS to provide users the most up to date weather information.

The Daily Business Planner

This planner application saves the user's input entered for each hour of the standard 9-5 business day. Local storage functionality remembers user input, and Moment.JS allows each time block to be color-coded based on the current time of day.
